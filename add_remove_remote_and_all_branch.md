### ==== see all the romote ===
`git remote -v`



### === add a new remote name "origin"
`git remote add origin git://new.url.here`


#### ====To remove remote use this:

`git remote remove `


#### ===Instead of removing and re-adding, you can do this:

`git remote set-url origin git://new.url.here`


### === see all the branch ===
`git branch -a`
